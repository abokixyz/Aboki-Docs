# Add Token Support

Configure which tokens your customers can receive on your platform. You can add custom tokens beyond the default ones (USDC, USDT, ETH, SOL).

## How It Works

1. **Business adds token** → Token becomes available for customers
2. **Set fee percentage** → You earn fees on each transaction  
3. **Enable/disable** → Control which tokens are active

## Authentication

You need a JWT token from your business login:


Authorization: Bearer Authorization: Bearer {your_jwt_token}



## Request Body Example

```json
{
  "network": "base",
  "tokens": [
    {
      "address": "0x833589fcd6edb6e08f4c7c32d4f71b54bda02913",
      "symbol": "USDC", 
      "name": "USD Coin",
      "decimals": 6,
      "feePercentage": 1.5
    }
  ]
}

Parameters

network: base, solana, or ethereum
address: Token contract address
symbol: Token symbol (USDC, CUSTOM, etc.)
name: Full token name
decimals: Token decimals (6 for USDC, 18 for most others)
feePercentage: Your fee (1.5 = 1.5%)
