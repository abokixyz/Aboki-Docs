# Add Custom Tokens

This endpoint allows you to add custom tokens to your platform.

## Authentication

You need a Bearer token from your business login:

```bash
Authorization: Bearer <your_jwt_token>

Request Body

{
  "network": "base",
  "tokens": [
    {
      "address": "0x833589fcd6edb6e08f4c7c32d4f71b54bda02913",
      "symbol": "USDC",
      "name": "USD Coin",
      "decimals": 6,
      "feePercentage": 1.0
    }
  ]
}

Example

curl -X 'POST' \
  'https://aboki-b2b-eobk.onrender.com/api/v1/business/tokens/add' \
  -H 'Authorization: Bearer <your_jwt_token>' \
  -H 'Content-Type: application/json' \
  -d '{
    "network": "base",
    "tokens": [
      {
        "address": "0x833589fcd6edb6e08f4c7c32d4f71b54bda02913",
        "symbol": "USDC",
        "name": "USD Coin",
        "decimals": 6,
        "feePercentage": 1.0
      }
    ]
  }'

Response

{
  "success": true,
  "message": "Custom tokens added successfully"
}
